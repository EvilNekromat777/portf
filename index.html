Ключевые навыки

- HTML5;

- CSS3 (Transform, Animate);

- Адаптивная верстка (знание особенностей верстки под мобильные устройства, планшеты, iOS, Android);

- Кроссбраузерность и валидность верстки как в десктопных, так и в мобильных браузерах;

- Умение верстать как desktop-first, так и mobile-first

- Flexbox, Grid Layout;

- Опыт работы в команде;

- Опыт понимания и редактирования чужого кода.

- Понимание и активное использование методологии БЭМ;

- Работа с макетом: Figma и PSD, Avocode, Perfect pixel;

- Работа с кодом: VSCode, PHPStorm;

- Сборка проекта: Gulp;

- Bootstrap 4, 5;

- Git (коммиты, ветвление веток, мерджи, пул реквесты, разрешение конфликтов);

- Препроцессор SASS (синтаксис SCSS);

- JavaScript (ES6);

- jQuery;

- Переменные, функции, циклы, условия, обработчики событий, добавление/удаление классов;

- Отслеживание клика по элементу/вне его, отслеживание скролла пользователя и привязка к нему анимации, воздействовие на содержимое DOM элементов;

- Подключение и стилизация слайдеров, аккордеонов, выпадающих списков, селектов - на уровне свободной кастомизации большинства популярных js-плагинов, таких как slick, fancy, jquery ui и т.д.
